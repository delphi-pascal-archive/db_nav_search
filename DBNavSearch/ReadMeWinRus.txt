   Компонент DBNavSearch для Delphi7.

   За основу взят стандартный DBNavigator из исходных текстов VCL Delphi7,
 из файла C:\Program Files\Borland\Delphi7\Source\Vcl\DBCtrls.pas выделено
 только то, что относится к компоненту DBNavigator и добавлена еще одна
 кнопка - nbSearch (Поиск). Также появилось еще одно опубликованное свойство
 (свойство видимое в Инспекторе) - RuKeybrd (логического типа), значение
 по умолчанию True, это свойство управляет автоматическим переключением
 клавиатуры на русскую раскладку при нажатии на кнопку nbSearch (Поиск) и
 появлении модального диалога для ввода значений полей искомой записи/записей.
 После закрытия модального диалога клавиатура переключается обратно на ту же
 раскладку, которая была до этого. Переключение работает корректно при любом
 количестве раскладок в системе (две и более) и даже при отсутствии русской
 раскладки - остается та же раскладка. Остальные опубликованные свойства
 компонента DBNavSearch такие же как и у стандартного DBNavigator. При
 включении показа всплывающих подсказок (ShowHint = True), всплывающие
 подсказки для кнопок компонента DBNavSearch выводятся на русском языке, также
 на русском языке выводятся запрос на подтверждение удаления записи и сообщение
 о том что искомая запись не найдена. Вся русификация компонента DBNavSearch
 инкапсулирована внутри модуля DBNAVSCH.PAS и никак не связана со строковыми
 ресурсами VCL.

   В свете клиент-серверных технологий в компоненте DBNavSearch сделаны еще два
 изменения по сравнению с DBNavigator:
   1. При нажатии на кнопку nbInsert (Вставка), выполняется Append,
      а не Insert. Соответственно изменена всплывающая подсказка на кнопку
      nbInsert с 'Вставить запись' на 'Добавить новую запись'.
   2. Если связанный с DBNavSearch DataSet является запросом (Query), то
      кнопка nbRefresh (Обновить данные) является неактивной (Enabled = False)
      вне зависимости от возможности редактировать данные (CanModify).

   Идея создания данного компонента взята из книги Кена Хендерсона 'Руководство
 разработчика баз данных в Delphi 2'. Все оригинальные файлы компонента
 DBNavSearch от Кена Хендерсона помещены в поддиректории Original (взяты с
 прилагаемого к книге CD).

   Представляемый компонент DBNavSearch для Delphi7 отличается от оригинального
 следующим:
   1. В диалог ввода значений для поиска включаются только видимые поля, т.е.
      поля у которых свойство Visible имеет значение True.
   2. Сам алгоритм поиска в оригинальном компоненте сводился к однократному
      вызову метода Locate класса DataSet. В представляемом компоненте вначале
      подготавливаются аргументы также, как и для метода Locate, но сам
      алгоритм поиска позаимствован из InterBase eXpress (IBX) файл
      IBCustomDataSet.pas (методы Locate и LocateNext), что позволило
      организовать поиск в цикле всех записей удовлетворяющих критерию поиска.
   3. Модифицирован модальный диалог ввода значений для поиска.

   В состав компонента входят следующие файлы:
      DBNAVSCH.DCR - пиктограмма для палитры компонентов,
      DBSEARCH.DFM - форма модального диалога ввода значений для поиска,
      DBNAVSCH.PAS - основной модуль компонента,
      DBSEARCH.PAS - модуль модального диалога ввода значений для поиска,
      DBNAV.RES    - ресурсы BITMAP для кнопок компонента.

   Установка компонента DBNavSearch в палитру компонентов Delphi7:
      1. Скопировать все файлы входящие в состав компонента в директорию
         C:\Program Files\Borland\Delphi7\Lib.
      2. В IDE Delphi7 открыть проект dclusr.dpk находящийся в той же
         директории.
      3. В форме проекта dclusr.dpk кликнуть кнопку Add, выбрать файл
         DBNAVSCH.PAS и затем кликнуть кнопку Install.
      4. Закрыть проект dclusr.dpk.
      5. В палитре компонентов (на закладке 'Data Controls') должен
         появиться новый компонент 'DBNavSearch'.

   Нигде в исходных текстах не проверяется версия Delphi, может быть компонент
 будет работать и в других версиях, кроме Delphi7, просто у меня установлена
 Delphi7 и поэтому я взял ее VCL за основу.

   Пример использования компонента (также с CD к книге Кена Хендерсона)
 находится в поддиректории SampleProject.
